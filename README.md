# MapGuessr

Google Maps ストリートビューを使った、場所当てゲームです。  
表示されたストリートビューの場所を地図上にピンで指定し、実際の位置との距離でスコアが決まります。  
**ソロプレイ専用**の練習・腕試し用ツールです。

---

## APIキーの設定

このゲームは **Google Maps JavaScript API キー**が必要です。  
APIキーを設定しないとゲームを開始できません。

### キーの取得

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. プロジェクトを作成し「Maps JavaScript API」を有効化
3. 「認証情報」→「APIキーを作成」でキーを発行
4. キーの制限（HTTP リファラーなど）を必要に応じて設定

### キーの入力・変更

- 初回起動時にセットアップ画面が表示されるので、取得したキーを入力して「保存して開始」
- キーはブラウザの **localStorage にのみ保存**されます（サーバーには送信されません）
- 変更するにはタイトル画面の「**APIキーを変更**」ボタンから再入力できます

---

## ゲームの流れ

GitHub Pages で公開しているページはこちらです。  
https://ermine27.github.io/MapGuessr/

1. ラウンド数・ゲームモード・制限時間を選んでスタート
2. ストリートビューに表示された場所を推理する
3. 右下のミニマップに**ピンをドロップ**して「Guess」ボタンを押す
4. 正解位置との距離とスコアが表示される
5. 全ラウンド終了後に結果画面でまとめてスコアを確認できる

---

## 得点について

- 1ラウンドの満点は **5,000点**
- 正解位置から離れるほど得点が下がります
- ピンを置かずにタイムアウトした場合は **0点**

---

## ゲームモード

| モード | 内容 |
|--------|------|
| **Move** | 自由に移動できる通常モード |
| **No Move** | 移動禁止。最初に表示された地点だけで判断する |
| **NMPZ** | No Move, No Pan, No Zoom。視点変更・ズームも禁止 |

---

## 制限時間

「Round Time」スライダーで 10秒〜10分 の範囲で設定できます。  
「制限なし」の場合は経過時間のみ表示されます。

---

## キーボードショートカット

ゲームプレイ中に使えるショートカットです。

| キー | 機能 |
|------|------|
| `Space` | Guess を確定 / 結果画面で次のラウンドへ進む |
| `Z` | 1手前の場所に戻る（Move モードのみ） |
| `C` | チェックポイントを設定 / チェックポイントに戻る（Move モードのみ） |
| `R` | スタート地点に戻る（Move モードのみ） |
| `N` | 北向きにアニメーションで回転（NMPZ 以外） |
| `↑ ↓ ← →` / `W A S D` | ストリートビューの視点移動（Move モードのみ） |
